# Adventure-Works-Sales-Analysis
This project focuses on analyzing Adventure works sales data from 2020 to 2022 for a sales store chain using SQL and Power BI. The goal is to derivekey business insights across sales, customers, products, geography, and orders. The dashboard analyzes three years of data and delivers actionable recommendations for business growth and optimization.
A combination of Power BI, Power Query, and DAX was used for data transformation, modeling, and interactive visualizations.
